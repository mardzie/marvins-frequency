# Substance is everything
Buzzwords won't save you.
In order to write good blog posts, you need to have knowledge about the topic.

# Different levels of experience

New variations make the deal.
If there isn't one for a language, write it.
But don't repeat the same topic too often.

Don't just write for noobs or experts.
The middle ground is under blogged.
